Instructions

You can use the machine to add tasks to the queue. The machine works by spinning a spinner. If the spinner lands on gold, it creates a video task; if it lands on black, it creates a slider task. Each time you make a choice, you can either run a new spinner from a panel of possible new spinners, or run the best spinner you've found so far.

To run the machine using the best spinner, click "run best spinner." The initial value of the best spinner is pre-set by the experiment.
